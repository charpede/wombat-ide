Wombat - Build {REVISION} ({DATE})

Source code:
- http://code.google.com/p/wombat-ide/

Features:
- Written in Java 1.6
- Deployed using WebStart (easy access and automatic updates)
- Built-in Scheme interpreter: Kawa 1.11 (http://sisc-scheme.org/, fully R5RS compliant)
- Detachable windows
- Configurable syntax highlighting
- Parenthesis matching
- Configurable automatic indentation and formatting

Known bugs:
- It's not possible to select regions of text (possibly because of parenthesis matching)
- Automatic indentation sometimes skips a line
- Reload options doesn't reload open documents
- Reload options doesn't change the Scheme that the REPL uses
